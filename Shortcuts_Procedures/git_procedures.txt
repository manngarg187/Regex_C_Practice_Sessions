
Git Notes:

— Steps to setup git in new folder/repo when you are working with multiple GitHub accounts:
1. Go to your project folder: cd /path/to/your/folder
2. Initialize the repo (only once): git init
3. Set your Git identity for this repo: git config user.name "Your Name" and git config user.email "your-email@example.com"
4. Stage all files: git add .
5. Commit the changes: git commit -m "Initial commit"
6. Connect to GitHub (choose one): Option 1 (first time): git remote add origin git@github.com-{identifier}/{repo-owner-name}/{repo-name}.git OR Option 2 (if remote already exists): git remote set-url origin git@github.com-{identifier}/{repo-owner-name}/{repo-name}.git
7. Verify the remote: git remote -v
8. Push to GitHub: git push -u origin main

— Steps to push the changes on GitHub when all the above setup already done:
1.  git add .
2. git commit -m “Name of the Commit”
3. git push
