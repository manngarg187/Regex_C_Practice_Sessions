

Steps to align code:
you can use the Command Palette to access formatting commands:
* Open the Command Palette: Ctrl + Shift + P (Windows/Linux) or Cmd + Shift + P (macOS).
* Type "Format Document" and select the corresponding command.

Here are the most useful keyboard shortcuts to move the cursor in the Mac terminal:

* Ctrl + A: Move to the beginning of the line
* Ctrl + E: Move to the end of the line
* Ctrl + F: Move forward (right) by one character
* Ctrl + B: Move backward (left) by one character
* Option (⌥) + Right Arrow: Move forward by one word
* Option (⌥) + Left Arrow: Move backward by one word
* Ctrl + U: Delete from the cursor to the beginning of the line
* Ctrl + K: Delete from the cursor to the end of the line
* Ctrl + W: Delete the word before the cursor
* Ctrl + D: Delete the character under the cursor

Go to definition	F12

🧠 Multi-Cursor & Selection
Action	Shortcut
